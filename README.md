# 前言



---

我是13年6月份毕业的一名大学生，到现在16年工作也有三年了，一直做的SEO这份工作，做一个行业做了有三年了，也算是有自己的一些优化心得，因此，准备写出来和大家共享一下，在下才疏学浅，有些的不对的地方或者自己理解偏错的地方希望各位有才华的读者指正。

我毕业那会（2013年），搜索引擎优化（Search Engine Optimization)还是非常热们的，这个也真没骗你们，我专业是通信工程，毕业后的第一份工作是SEO，想起来我也是觉得诧异，那个时候，班里找到工作的都有一大半了，临近毕业的我也是十分的着急，觉得

* 第一：是很迷茫，不知道要往什么方向发展，行业这么多，我该去怎么寻找更适合我的方向呢？
* 第二：是很不自信，因为自己肚里好像什么知识都没有。
* 第三：是很焦虑，越到毕业的日子临近，越是焦虑，不知何去何从，如何去走。

之后的，也就上网了解以下当今时段热门的工作，加了几个工作的qq群，在里面也咨询了一些已经就业的前辈。
也就是在那个时候，接触到了SEO（可见当时真的非常热们）。
我SEO是自学成才的，刚一开始是从网上看的视频教程，那种很基础的入门课程，课程里面主要讲解一些

* 基本概念
* 互联网现状
* 搜索引擎市场
* SEO优化基本操作
* 标签（标题，描述，关键词等等）基本标签的撰写
* 关键词选取（比如指数）和网站定位

说实话，现在来看，或许这些都是非常非常基础的入门知识，但，当时的我，学的时候真是费了不少苦心才把这些概念和基础知识消化掉，说实话，基础搭打的牢，比什么都要好。

后来，看了之后给与我SEO收益最大的一本书，我觉得，有经验的SEO人员都会推荐这本书，ZAC写的《SEO实战密码》，或许读者对这本书褒贬不一，有人奉为枕边书，有人觉得不实用（因为里面大多都是讲谷歌搜索引擎的优化操作的，而国内普遍是百度，并且书上写的工具大多都是针对谷歌的，国内搜索引擎涉及到的很少）
